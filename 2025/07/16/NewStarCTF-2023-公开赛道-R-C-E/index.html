但是这里利用到的是sed命令未被过滤，然后修改index.php的代码然后绕过正则

sed（Stream Editor）是一个强大的文本流处理工具，常用于快速修改文件、提取数据或自动化处理。

常用选项：

  • -i：直接修改文件（慎用，建议先不加 -i 测试）
  • -n：抑制默认输出（常与 p 命令配合）
  • -e：连接多个命令（如 sed -e 's/foo/bar/' -e 's/hello/world/'）
  • -r：启用扩展正则表达式（类似 grep -E）

我们还需要知道sed有哪些命令

命令 功能 语法示例 作用说明 CTF 应用场景
s 替换文本 sed 's/旧/新/g' file 全局替换匹配的文本 删除敏感字符（如 ` 、&`）
d 删除行 sed '3d' file 删除第 3 行 清理日志中的干扰行
sed '/error/d' file 删除含 error 的行 隐藏攻击痕迹
p 打印行 sed -n '5p' file 打印第 5 行（需 -n 抑制默认输出） 提取 flag{.*} 格式数据
sed -n '/flag/p' file 打印含 flag 的行
a 行后追加 sed '2a\新增内容' file 在第 2 行后追加文本 插入恶意代码或配置
i 行前插入 sed '/pattern/i\插入内容' file 在匹配行前插入文本 篡改关键配置
c 整行替换 sed '4c\替换内容' file 将第 4 行完全替换 修改文件关键逻辑
r 插入外部文件内容 sed '3r external.txt' file 在第 3 行后插入外部文件内容 合并隐藏数据
w 写入到文件 sed '/flag/w output.txt' file 将含 flag 的行写入 output.txt 提取关键数据到新文件
y 字符转换 sed 'y/abc/ABC/' file 将 a→A、b→B、c→C（逐字符替换） 简单编码/解码
q 退出处理 sed '5q' file 处理到第 5 行后退出 快速截断大文件
