var ip = req.connection.remoteAddress; 这种方式获取客户端 IP 之所以难以通过 HTTP 请求头伪造，是因为它直接从 TCP 连接层 获取 IP 地址，而不是从 HTTP 头部（如 X-Forwarded-For）读取。

req.connection.remoteAddress（或 req.socket.remoteAddress）是 Node.js 从 TCP 连接 直接获取的客户端 IP 地址，属于 传输层（OSI 第4层） 的信息，而不是应用层（HTTP 头部）。
